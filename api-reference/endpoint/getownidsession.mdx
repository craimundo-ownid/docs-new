---
title: "POST /getSessionByLoginId"
description: "The OwnID server sends a request after the user has successfully logged in with OwnID, expecting to get a session object"
---

### Request Body

<ParamField body="loginId" type="string" required>
  The unique identifier of a user in the database, usually email or phone.
</ParamField>

<ParamField body="sessionType" type="string">
  Session type, browser or mobile.
</ParamField>

### Expected Response

<ResponseField name="token" type="object">
  When the OwnID makes a request, it expects to receive back a `token` as a JSON object, often a JWT.
</ResponseField>

<RequestExample>
```bash Example Request
curl --location --request POST 'https://bazco.company/getSessionByLoginId' \
--header 'Content-Type: application/json' \
--data-raw '{
   "loginId":"sol@testmail.com",
   "sessionType":"browser",
}'
```
</RequestExample>
